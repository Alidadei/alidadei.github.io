# 📅 每日追踪流程

> 基于《早期 AI 初创公司识别指南》第三节的"每周 30 分钟扫描"流程优化版。
>
> **核心原则：每周 30 分钟，雷打不动。比 PR 早 3-6 周看到信号。**

---

## 每周固定节奏

```
┌─────────────┬──────────────────────────────────────┬────────┐
│    周一     │ HF Trending + arXiv 周推送           │ 15min  │
├─────────────┼──────────────────────────────────────┼────────┤
│    周三     │ VC Portfolio 更新 + GitHub 信号      │ 10min  │
├─────────────┼──────────────────────────────────────┼────────┤
│    周五     │ Twitter/X 研究员动向 + 信号表更新    │ 5min   │
└─────────────┴──────────────────────────────────────┴────────┘
                                          每周总计 ≤ 30min
```

---

## 周一：模型 & 学术信号（15min）

### 步骤 1：Hugging Face Trending（5min）

打开以下页面，看前 10 名中有没有值得关注的：

- **Models Trending**：https://huggingface.co/trending
- **Spaces Trending**：https://huggingface.co/spaces/trending

**重点看**：
- 下载量曲线是否**非线性陡升**（不是缓慢线性增长才说明爆了）
- 模型卡上的**作者机构**（org 标识）——第一次出现的新 org 值得记下来
- **中文团队发布的模型**：留意发布机构是否跟我们追踪列表里的公司匹配

### 步骤 2：arXiv 周推送（10min）

订阅/打开以下分类的最近一周论文：

- **cs.CL**（计算语言学）：https://arxiv.org/list/cs.CL/recent
- **cs.LG**（机器学习）：https://arxiv.org/list/cs.LG/recent

**快速滤出高信号论文**：
1. 看标题：出现 `MoE`、`long-context`、`memory`、`agent`、`world model`、`efficient` 等关键词的论文
2. 看作者单位：是否来自我们追踪的初创公司？
3. 看论文类型：technical report > 理论论文（创业公司倾向发 tech report）

> **技巧**：用 Semantic Scholar 查某篇论文发出后 30 天内的被引次数——一周内被引 10+ 次的几乎可以确定是热门方向。
> 链接：https://www.semanticscholar.org/

### 记录模板

```markdown
## 周一扫描 - YYYY-MM-DD

### HF Trending 观察
- [模型名] 由 [机构] 发布，下载量 [数字]，排名第 [N]
- 关注点：________________________________

### arXiv 观察
- [论文标题] — 关键词：[关键词]
- 作者单位：[机构]
- 被引信号：[快速查一下引用量]

### 信号表更新
- [公司名]：新信号 → 评级调整建议
```

## 周三：融资 & 工程信号（10min）

### 步骤 1：VC Portfolio 扫描（5min）

**国内 VC（关注 AI 方向的 portfolio 页）**：
- 红杉中国：https://www.sequoiacap.com/china/companies/
- 真格基金：https://zhenfund.com/portfolio/
- 高瓴创投：https://www.hillhousecap.com/
- 五源资本：https://www.5ycap.com/
- 砺思资本（Monolith）：https://www.monolithmanagement.com/

**快速扫**：看 portfolio 页最近是否有新增的 AI 公司。

### 步骤 2：跟踪公司的 GitHub 信号（5min）

对当前追踪列表里 **🔥 重点关注** 的公司，查其 GitHub org：

1. 看 **14 天内 commit velocity**：涨 200%+ 通常意味着刚关完一轮融资 + 大规模招人
2. 看 **contributor 数变化**：涨 50%+ 也是强信号
3. 看 **star 增速**：用 star-history.com 看曲线形状

> **参考工具**：
> - https://www.star-history.com/ — 看 star 增速曲线
> - GitHub Trending：https://github.com/trending 加自定义关键词过滤

## 每天 5 分钟"快速扫描"（可选）

如果某天特别忙、但不想完全断掉，做这个最小版本：

```
1. 打开 HF Trending（1min）— 看有没有新模型冲榜
2. 打开一条关注公司的新闻/融资推送（2min）
3. 打开信号表瞄一眼（2min）— 提醒自己这周在跟踪什么
```

## 季节性的额外投入

- **每月 1 次**：扫一遍 VC portfolio 页，看有没有新增投资
- **每季度 1 次**：看当期 NeurIPS/ICML/ICLR 的 Oral/Spotlight 论文作者去向——很多人发完论文跳去初创公司
- **有大会时**：WAIC（7月）、智源大会、Demo Day——集中扫

---

## 降级/移除规则

| 情形 | 操作 |
|------|------|
| 连续 4 周无任何新信号 | 从 🔥 降至 👀 |
| 连续 8 周无任何新信号 | 从 👀 降至 📡 或移除 |
| 估值/融资进入后期轮次 | 移入 ✅ 已成熟 |
| 出现红旗清单中的问题 | 立即降级并记录原因 |
| 公司倒闭/方向 pivot | 移除并记一笔经验 |

## 红旗清单（快速参考）

从框架中摘出来的，每次评估时对照：

- [ ] "我们的 AI" 实际上是别人 API 的封装？
- [ ] 没有任何技术博客 / 论文 / 开源仓库？
- [ ] 创始人说不清自己押的是什么技术 bet？
- [ ] 数据来源说不清（合规风险）？
- [ ] 核心团队 6 个月内有人离职潮？
- [ ] 估值和实际产品/收入严重脱钩？
- [ ] 创始人背景只有"连续创业者"而无技术分量？
- [ ] JD 用语充满空话（"最先进的 AI 技术"/"拥抱变化"/"AGI 时代"）？
