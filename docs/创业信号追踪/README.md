# 🚀 AI 创业公司信号追踪系统

> 基于《早期 AI 初创公司识别指南》框架的实战追踪工具。
> 
> 目标：比大多数人早 3-12 个月看到信号簇，在爆红之前识别有潜力的 AI 初创公司。

---

## 核心方法

来自《早期 AI 初创公司识别指南》的 **6 维度信号框架**：

| 维度 | 关键问题 |
|------|----------|
| ① **创始人背景** | 是否有"非随机的过去"？顶级实验室 / 顶级竞赛 / 上一段已成功？ |
| ② **技术 bet 清晰度** | 能否用一句话说清具体押什么方向？还是"做通用 AGI"？ |
| ③ **首批投资人信号** | Tier-1 机构领投？产业资本跟投？还是只有财务投资？ |
| ④ **工程/开源信号** | GitHub commit velocity、HF 榜单、arXiv 论文引用是否异常上升？ |
| ⑤ **招聘/组织信号** | JD 用语是否具体？组织设计是否反常（扁平、PBC、长期权）？ |
| ⑥ **产品/商业化信号** | 有真实客户吗？API 价格异常低？Demo 比同行明显好？ |

**单条信号不够，多个维度同时冒出才形成"信号簇"——才是行动的时点。**

## 每日追踪节奏

| 时间 | 内容 | 时长 |
|------|------|------|
| **周一 15min** | HF Trending + arXiv cs.CL/cs.LG 周推送 | 15min |
| **周三 10min** | 扫三家 Tier-1 VC 的 portfolio 更新 + 关注的早期公司 GitHub commit velocity | 10min |
| **周五 5min** | 扫盯的 10-20 个研究员 Twitter/X，看转发了谁 + 更新信号表状态 | 5min |

每天完成后在 `日志/YYYY-MM-DD.md` 记录观察，全流程每周约 30 分钟。

## 追踪优先级

- **🔥 重点关注**：多个强信号同时出现，技术 bet 清晰，还在早期窗口期
- **👀 持续观察**：有 1-2 个信号值得注意，需要更多验证
- **📡 雷达扫描**：刚冒头或还在非常早期，保持最低频率跟踪
- **✅ 已成熟**：估值已高、信息已充分反映，不再属于早期跟踪范围
