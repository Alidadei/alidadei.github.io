# 🧠 每日推荐系统

> 基于信号表 + 每日观察，自动（或半自动）生成"今天该关注什么"的推荐。
>
> 核心逻辑：**当多个维度同时出现信号时，就是该行动的时候。**

---

## 推荐算法逻辑

每次扫描完成后，对信号表中的每家公司运行以下规则：

```
推荐优先级 = Σ（各维度新信号强度）× 当前评级权重 + 临近事件触发
```

### 权重表

| 变量 | 权重 | 含义 |
|------|------|------|
| 新强信号（⭐新增） | +3 | 首次出现一个维度的强信号 |
| 新中信号（新增） | +2 | 首次出现一个维度的中信号 |
| 信号升级（⭐→⭐⭐⭐） | +2 | 现有信号增强 |
| 连续 4 周无信号 | -3 | 进入静默期 |
| 出现红旗 | -5 | 立即警告 |
| 🔥 关注级 | ×1.5 | 信号乘数 |
| 临近事件（大会/报告发布） | +2 | 增加临时权重 |

### 推荐输出格式

```
🔥 今日重点关注：[公司名]
  └─ 理由：[新信号描述 + 关联维度]
  └─ 建议动作：[查 GitHub / 读论文 / 看融资新闻]

👀 值得一看：[公司名]
  └─ 新动态：[简述]

⚠️ 需要注意：[公司名]
  └─ 红旗：[具体问题]
```
