# 📋 信号追踪规则

> 定义信号等级、评级调整规则、升级/降级/移除标准。
> 确保追踪系统的一致性——不管谁来维护，同一家公司得到相同的评估。

---

## 1. 信号强度定义

### ⭐⭐⭐ 强信号（3 分）

有**公开可验证的事实**支撑：

| 维度 | 强信号示例 |
|------|------------|
| 创始人背景 | 顶级实验室（DeepMind/FAIR/OpenAI/Brain）+ 论文被广泛引用；IOI/IMO 金牌；上一段已成功退出 |
| 技术 bet | 能用一句话说清具体方向，有论文或产品证明；方向是反共识的 |
| 投资人 | Tier-1 机构（红杉/高瓴/真格/IDG/砺思/Sequoia/a16z/Lightspeed）+ 产业资本跟投 |
| 工程/开源 | GitHub commit 速度骤升（14 天 200%+）；HF 登顶榜；论文被引异常高 |
| 招聘/组织 | JD 出现具体技术词（MoE/RLHF/长上下文/KV cache）+ 引用自家论文 |
| 商业化 | 有 1-2 个有名有姓的企业客户 + 正毛利 / 可观的 ARR |

### ⭐⭐ 中信号（2 分）

有**迹象但需要更多证据**：

| 维度 | 中信号示例 |
|------|------------|
| 创始人背景 | 大厂中高层（非核心研究岗）；有技术背景但无顶级实验室经历 |
| 技术 bet | 方向较具体但还在早期，无公开论文或产品 |
| 投资人 | 知名机构参与但非领投；只有财务投资方无产业方 |
| 工程/开源 | GitHub 有提交但不密集；HF 在榜但不在前列 |
| 招聘/组织 | JD 有一些技术词但不具体 |
| 商业化 | 有少量客户但名字不公开；有收入但数额不清 |

### ⭐ 弱信号 / 信息不足（1 分）

仅靠基本信息，**不足以做出判断**。大多数新出现的公司从这个分数起步。

## 2. 综合评级

从信号表中汇总 6 维度的得分：

| 总分 | 评级 | 含义 |
|------|------|------|
| **14-18** | 🔥 重点关注 | 多个强信号同时出现，值得深入跟踪。每 2 周详细看一次 |
| **10-13** | 👀 持续观察 | 有 1-2 个强信号或多个中信号，需要在信号表里记录进展 |
| **4-9** | 📡 雷达扫描 | 刚出现或信号弱，保持最低频率观察。每月扫一眼就行 |
| - | ✅ 已成熟 | 不再早期跟踪，仅做参考 |

## 3. 升级/降级触发条件

### 升级条件（满足任一即可申请升级）

| 从 | 到 | 条件 |
|----|----|------|
| 📡 → 👀 | 出现 1 个强信号或 2 个中信号 | 例如：任何一家公司拿到 Tier-1 投资 |
| 👀 → 🔥 | 出现另一个强信号，总维度 ≥3 个有强信号 | 例如：拿到投资后技术报告也发了 |
| 🔥 → 行动 | 信号簇完整：创始人 + 技术 + 投资人 + 工程/商业化 ≥4 个强信号 | 例如：框架中的"该投简历的时候" |

### 降级条件（满足任一立即降级）

| 从 | 到 | 条件 |
|----|----|------|
| 🔥 → 👀 | 连续 4 周无任何新信号 | 公司进入了沉默期 |
| 👀 → 📡 | 连续 8 周无任何新信号 | 或出现红旗清单中的问题 |
| 📡 → 移除 | 连续 12 周无信号 + 无任何消息 | 或公司关闭/停止运营 |
| 任何 → ✅ | 完成后期轮次融资（D 轮+/估值超 $5B） | 信息已充分反映在市场中了 |

### 红旗触发条件（**立即降级或移除**）

- "我们的 AI" 是某开源模型的封装
- 核心团队 6 个月内出现离职潮
- 创始人公开言论与公司技术能力严重不符
- 被曝数据合规问题
- 公司陷入严重财务危机

## 4. 评分实操指南

### 信息源优先级

1. **一手信息优先**：论文全文、技术报告、GitHub 仓库、融资公告 > 媒体报道
2. **二手信息标注来源**：使用报道信息时在备注栏标注来源链接
3. **谣言/猜测不录入**：只有公开可验证的信息才计入信号

### 评分时的注意

- **信号会过期**：一条"去年拿了大额融资"的信息，到今年信号价值已大幅衰减
- **信号要对比**：同一赛道的公司之间对比——面壁智能 GitHub 3800 万下载是一个信号，但需要跟同行比
- **不重复计分**：同一件事的表现形式不同（如融资新闻 → 投资人 + 融资速度），只取强的一面

### 更新频率建议

| 公司状态 | 建议更新频率 |
|----------|-------------|
| 🔥 重点关注 | 每 2 周重新评估一次 |
| 👀 持续观察 | 每月重新评估一次 |
| 📡 雷达扫描 | 每季度扫一眼即可 |
| ✅ 已成熟 | 有重大新闻时更新 |

## 5. 信号表字段说明

| 字段 | 说明 |
|------|------|
| **公司名** | 中文名 + 英文名 / 品牌名 |
| **维度信号** | 6 维度各 ⭐×3 评分 + 说明 |
| **综合** | 总分/18 + 一句话评价 |
| **跟踪日志** | 链接到 `record/` 下的观察日志 |
| **关注理由** | 为什么这家公司值得跟踪 |
| **更新日期** | 最近一次评估日期 |
