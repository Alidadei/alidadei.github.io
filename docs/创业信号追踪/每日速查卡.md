## 信号强度速判

| 信号 | 一眼识别法 |
|------|-----------|
| ⭐⭐⭐ | 公开数据 + 可验证 + 异常值 |
| ⭐⭐ | 有迹象但不够 |
| ⭐ | 听说有这家公司 |

**法则**：3+ 个 ⭐⭐⭐ 在不同维度 = 信号簇 = 行动信号
