# 网站制作感悟

这是一个非常需要时间和耐心的事情，不建议在有正事的时候做，我认为它更适合在家里边喝茶边做。整个过程中，我深切体会到让后端+AI去替代前端是多么荒谬：因为对于一个完全没有前端经验的人来说，这样的工作量真的不小。

## 审美

审美这个东西很看个人的天赋和灵感，想搓出不一样的界面，得真的在意网页的颜值，并对其有较高的要求和苛刻的眼光。

尝试了DESIGN.md，但是最后发觉还是不如自己一点一点抠细节，前端审美真的是很吃细节的东西，若真的想做出不千篇一律的网站页面，还是按照个人审美喜好去一点点调整的好。

## 写作

目前AI写出来的东西还是没法读的，目前来看，AI依然无法替代人类写出个性化、多元化的高质量文字作品。

## TIPS

机械重复的流程（比如压缩图片）及时让agent沉淀成一个脚本，不要什么改动都让agent去搞。

架构最好一开始就确定清晰，不要在老的架构上开新的架构。

前端UI设计一大难点在于agent不清楚你要改的具体是哪个元素，1可以通过某些专门的设计网站查找清楚，2可以通过一个视觉AI帮你弄明白。
