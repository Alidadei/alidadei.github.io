# 国内 AI 初创公司潜力清单（工作草稿 · 待核实）

> ⚠️ **先读这一段**：本清单**没有基于实时搜索**。整理它时，web-search / web-reader 的调用额度已耗尽（2026-07-28 重置），brave-search 网络不通，内置 WebFetch 被环境拦截——所有实时通道都不可用。
>
> 因此本文件**不是带引用的调研**，而是 AI 基于截止 **2025 年初** 的训练知识、套用 [《早期AI初创公司识别指南》](早期AI初创公司识别指南.md) 第二节的"领先信号清单"做的一份**工作草稿**。每家公司只陈述相对稳定的公开事实（创始人背景、技术方向、已知投资人类型），**具体金额、估值、日期一律不写死**，模糊项标注「待核实」。
>
> 正确用法：把这份当作"候选名单起点"，用调研文档第三节的 7 个信号源逐家核实，**不要直接当结论用**。

---

## 阅读说明

本清单用调研文档第二节 checklist 的四个最硬维度做速评：**①创始人"非随机过去" ②技术 bet 清晰度 ③首批投资人 ④开源/论文/工程产出**。

分四档，对应调研里"被识别程度"的递进：

- **A 档**：头部独角兽，已过早期——识别窗口基本关闭，对照 DeepSeek / 智谱现在的状态。
- **B 档**：中早期 + 清晰 bet + 顶级背景——**最值得关注**，对照月之暗面早期。
- **C 档**：具身智能赛道蜂拥——对照宇树 2016-2020，需精挑，红旗多。
- **D 档**：AI 编程 / 应用层——简述，不下断言。

调研文档已覆盖的 DeepSeek、智谱、月之暗面、宇树**不重复列**。

## 一、A 档：头部独角兽（识别价值降低，但仍在爬升）

### MiniMax（名之梦）

- **创始人非随机过去**：闫俊杰，商汤科技前副总裁（研发方向）。2021 年创立。
- **押的 bet**：多模态（语音 / 视频生成 / 角色扮演 AI）；海外产品 **Talkie**（角色聊天）走通出海，国内有海螺 AI。
- **已知信号**：阿里、腾讯、红杉中国、高瓴、IDG、MIH 等参投；2024 年推出视频生成模型。
- **⚠️ 待核实**：最新一轮估值与轮次；Talkie / 海螺的当前用户数据。
- **速评**：①✓ ②✓（多模态 + 出海，bet 清晰）③✓。**但估值已高，早期窗口已过**——按调研判断标准，加入回报的"信号价值"在下降。

### 百川智能（Baichuan AI）

- **创始人非随机过去**：王小川，搜狗创始人 / 前 CEO，清华出身。2023 年 4 月创立。
- **押的 bet**：通用大模型起家，2024 年起**反共识地转向医疗 AI**（Baichuan-Medical）。
- **已知信号**：阿里、腾讯、小米、深创投、清华系投资；Baichuan 系列早期开源。
- **⚠️ 待核实**：最新估值；医疗 bet 的商业化落地进展。
- **速评**：①✓（"上一段已赢过一次"= 搜狗）②✓（垂直医疗是反共识）。估值高，窗口部分关闭。

### 阶跃星辰（StepFun）

- **押的 bet**：多模态大模型（Step 系列，语言 → 视觉 → 视频），业内视为国内多模态最激进的玩家之一。
- **⚠️ 待核实**：**创始人背景**（AI 不在此写不确定的姓名，这点必须查清才能用"非随机过去"维度评判）、最新融资与估值。
- **已知信号**：上海国资、腾讯、五源等（待核实）；2024 年多模态产品发布密集。
- **速评**：②✓（多模态 bet 清晰）。①待核实——这是用调研框架评判它**最关键的一块拼图**，务必先查创始人。

### 零一万物（01.ai）

- **创始人非随机过去**：李开复，创新工场董事长，前 Google / 微软高管。2023 年创立。
- **押的 bet**：Yi 系列开源大模型 + 应用层。
- **🚩 关键风险（重点核实）**：2024 年末有公开报道公司**缩减 / 拆分预训练、转向应用层**（李开复方面另有表述）。这正好命中调研文档"红旗"里的"核心团队 / 战略不稳"——**这是本清单里最需要核实 2025-2026 最新状况的一家**。
- **速评**：①✓（顶级）②✓（Yi 开源口碑）但有重大战略调整红旗。

## 二、B 档：中早期 + 清晰 bet + 顶级背景（最值得关注）

### 面壁智能（ModelBest）

- **创始人非随机过去**：**刘知远**（清华大学副教授，NLP）联合创始人；**李大海**任 CEO（知乎前 CTO）。
- **押的 bet**：**端侧小模型**（MiniCPM 系列），反共识地押"小模型 / 手机端部署"，与行业"越大越好"反向——典型的"一句话讲清具体押什么"。
- **已知信号**：MiniCPM 开源、在手机端跑通；华为哈勃、知乎等投资。
- **⚠️ 待核实**：商业化进展、最新融资。
- **速评**：①✓ ②✓（反共识且具体）④✓（开源产出）——**信号簇最齐全的一家，对照文档"月之暗面早期"那一档，建议优先跟踪**。

### 潞晨科技（ColossalAI）

- **创始人非随机过去**：**尤洋**，UC Berkeley 博士 / 博士后（高性能计算方向，师从 James Demmel），后回国创业。
- **押的 bet**：大模型**训练 + 推理 infra / 效率**（反共识押"效率"而非"更大模型"），ColossalAI 开源训练框架；后扩展到推理优化与视频生成。
- **已知信号**：ColossalAI 在 GitHub 有较高 star；学术论文持续产出；开源社区活跃。
- **⚠️ 待核实**：商业模式（infra 公司怎么赚钱）、最新融资、团队规模。
- **速评**：①✓（顶级实验室 + 学术）②✓（效率 bet，对照文档 Mistral 的 compute-efficient 路径）④✓。

### 生数科技（Vidu）

- **创始人非随机过去**：**朱军**，清华大学教授（人工智能研究院 / 机器学习方向）。
- **押的 bet**：视频生成（**Vidu**），2024 年发布，被视为"中国 Sora"之一；底层是扩散 + Transformer。
- **已知信号**：百度、锦秋基金等投资；2024 年 Vidu 发布时受广泛关注。
- **⚠️ 待核实**：模型最新效果对比、最新融资。
- **速评**：①✓ ②✓。**但赛道极度拥挤**（快手可灵、阿里、字节、腾讯都在做），竞争是最大变量。

### 硅基流动（SiliconFlow）

- **创始人非随机过去**：**袁进辉**，OneFlow（一流科技，深度学习框架）创始人，清华博士出身。
- **押的 bet**：大模型**推理 API 云服务**，聚合 DeepSeek / Qwen / Llama 等开源模型，主打"开发者用得起、推理便宜"。
- **已知信号**：借 DeepSeek 爆发的东风，API 调用量陡增；做了开源模型聚合站。
- **⚠️ 待核实**：最新融资、营收规模、毛利率。
- **速评**：①✓（相邻领域已赢过一次 = OneFlow）②✓。**但护城河依赖上游开源模型 + 价格战**，需重点看毛利率与差异化（调研文档的"产品 / 价格信号"维度）。

## 三、C 档：具身智能赛道（2024-2025 蜂拥，需精挑）

> 这个赛道目前**红旗密度最高**：大量公司估值爬升远快于产品 / 收入，命中调研文档"估值与产品脱钩"红旗。务必逐家用 checklist 核。

### 智元机器人

- **创始人非随机过去**：**彭志辉**（网名"稚晖君"），前华为"天才少年"，B 站知名科技 up 主（百万粉丝级），2023 年初离职创业。
- **押的 bet**：通用人形机器人（远征系列），具身智能。
- **已知信号**：高瓴、百度、腾讯、上汽等投资（待核实最新轮）；稚晖君本人的技术 IP = 天然招聘磁铁 + 传播信号。
- **⚠️ 待核实**：量产进展、最新估值、商业化收入。
- **速评**：①✓（华为天才少年 + 技术 IP，非随机的过去）②✓。对照文档"创始人有传播力 + 反共识方向"的信号簇。

### 银河通用（Galbot）

- **押的 bet**：具身大模型 + **轮式双臂机器人**（押"轮式 + 操作"，区别于清一色人形），主抓通用操作 / 分拣场景——bet 有差异化。
- **已知信号**：美团、北汽等产业资本投资；与美团等探索物流 / 零售场景（有名有姓的客户，对照文档"客户结构"维度）。
- **⚠️ 待核实**：**创始人背景**（具身智能圈，具体姓名 AI 不写死）、最新融资。
- **速评**：②✓（轮式差异化）③✓（产业资本 = 有场景背书）。①待核实。

### 星动纪元（Robot Era）

- **押的 bet**：人形机器人 + **VLA（视觉-语言-动作）模型**，bet 贴合前沿。
- **已知信号**：联想创投等投资；2024-2025 人形产品迭代。
- **⚠️ 待核实**：**创始人背景**（清华系关联，具体待核实）、最新动态。
- **速评**：②✓（VLA 前沿）。①待核实。

## 五、怎么核实这份清单（直接套调研文档第三节）

对清单里**任何一家**，按这 5 步走，2 小时能把一家核到可用程度：

1. **工商 + 融资**：IT 桔子 / 天奶查 / 天眼查——成立时间、股东、历轮融资与投资方（验证 ③ 首批投资人）。
2. **开源 / 工程产出**：GitHub org 看 commit velocity + contributor 增速；Hugging Face 看模型下载曲线（验证 ④）。
3. **招聘 JD 用语**：直接读全文——出现 MoE / VLA / KV cache / 具体 benchmark = 真前沿；只有"最先进的 AI 技术"= 套壳红旗（文档信号源 6）。
4. **创始人近一年公开表态**：尤其战略是否调整（零一万物的红旗就是这步能抓到）。
5. **技术圈口碑**：机器之心 / 量子位 / 即刻 / 小红书 AI 圈——比 PR 稿早 1-2 个月（文档信号源 5）。

**本次最该优先核实的三件事**：
- 零一万物 2025-2026 的战略稳定性（红旗项）
- 阶跃星辰、银河通用、星动纪元的**创始人背景**（清单里 ① 维度的缺口）
- 各家**最新一轮融资**（清单里 ③ 维度全部待更新）

## 六、不确定性与红旗（必读）

- **时效缺口**：本清单信息截止 **2025 年初**。2025 下半年 → 2026-07 之间的动态（新融资、战略转向、产品迭代、人事变动）AI **全部不知道**，必须核。
- **无引用**：本文件不是 deep-research 产物，没有可点击来源——和焕那份带 27 条引用的调研**严谨度不同级**，别混用。
- **样本偏向"已被关注"**：A/B/C 档基本都是圈内已知道的名字。真正"早期未被识别"的标的（调研文档的核心目标），需要焕自己用第三节方法论去 GitHub / HF / arXiv 扫——这份清单帮不到那一层。
- **具身智能整体过热**：C 档多家估值远超产品进度，**默认带红旗**，别因为"赛道火"就放松 checklist。
